**Step 2:** Implement `pnpg/pipeline/geo_enricher.py`:

**Imports:**
```python
import logging
import os
import time
import geoip2.database
import geoip2.errors
import maxminddb
```

**Module-level state:**
```python
logger = logging.getLogger(__name__)
SECONDS_IN_DAY = 86400

_country_reader: geoip2.database.Reader | None = None
_asn_reader: geoip2.database.Reader | None = None
```

**open_readers(country_db_path: str, asn_db_path: str) -> None:**
- Set `global _country_reader, _asn_reader`
- Try to open `geoip2.database.Reader(country_db_path, mode=maxminddb.MODE_MEMORY)` — if FileNotFoundError, log WARNING "GeoLite2-Country database not found at {path}", set `_country_reader = None`
- Try to open `geoip2.database.Reader(asn_db_path, mode=maxminddb.MODE_MEMORY)` — if FileNotFoundError, log WARNING "GeoLite2-ASN database not found at {path}", set `_asn_reader = None`
- Log INFO with count of readers opened

**close_readers() -> None:**
- If `_country_reader` is not None: `_country_reader.close()`
- If `_asn_reader` is not None: `_asn_reader.close()`
- Set both to None

**enrich_geo(event: dict) -> dict:**
- Extract `ip = event.get("dst_ip")`
- Initialize `country, asn, org = None, None, None`
- If ip and `_country_reader` is not None:
  - Try: `resp = _country_reader.country(ip)`, `country = resp.country.iso_code`
  - Except `(geoip2.errors.AddressNotFoundError, Exception)`: `country = None`
- If ip and `_asn_reader` is not None:
  - Try: `resp = _asn_reader.asn(ip)`, `asn = f"AS{resp.autonomous_system_number}"`, `org = resp.autonomous_system_organization`
  - Except `(geoip2.errors.AddressNotFoundError, Exception)`: `asn, org = None, None`
- Return `{**event, "dst_country": country, "dst_asn": asn, "dst_org": org}`

**check_db_freshness(db_path: str, max_age_seconds: float, metric_key: str) -> None:**
- Try: `age = time.time() - os.path.getmtime(db_path)`
  - If `age > max_age_seconds`: `logger.warning("%s: database at %s is %.1f days old (limit: %.1f days)", metric_key, db_path, age / SECONDS_IN_DAY, max_age_seconds / SECONDS_IN_DAY)`
- Except FileNotFoundError: `logger.warning("%s: database file not found at %s", metric_key, db_path)`

Run all tests until GREEN. Verify existing tests still pass.
