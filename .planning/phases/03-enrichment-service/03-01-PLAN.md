from collections import OrderedDict

**TtlLruCache class:**
- `__init__(self, maxsize: int, ttl: float)` — store maxsize, ttl, create `self._cache: OrderedDict = OrderedDict()`, create `self._lock = Lock()`
- `get(self, key: str) -> str | None`:
  - Acquire `self._lock`
  - Look up `self._cache.get(key)` — return None if missing
  - Unpack `(value, expires_at)` from entry
  - If `time.monotonic() > expires_at`: delete key from cache, return None
  - Call `self._cache.move_to_end(key)` to mark as recently used (LRU)
  - Return `value`
- `set(self, key: str, value: str) -> None`:
  - Acquire `self._lock`
  - If key already in cache: `self._cache.move_to_end(key)`
  - Elif `len(self._cache) >= self.maxsize`: `self._cache.popitem(last=False)` (evict LRU)
  - Store: `self._cache[key] = (value, time.monotonic() + self.ttl)`
