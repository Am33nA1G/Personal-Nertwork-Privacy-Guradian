    1. Create `pnpg/capture/queue_bridge.py` per RESEARCH.md Pattern 2 (CAP-05, CAP-06, CAP-07, CAP-08):

       `make_packet_event(pkt) -> dict`:
       - Returns `{"timestamp": datetime.datetime.now(datetime.timezone.utc).isoformat(), "monotonic": time.monotonic(), "raw_pkt": pkt}`
       - The `timestamp` field satisfies CAP-08 (ISO8601 wall-clock)
       - The `monotonic` field satisfies CAP-08 (monotonic for ordering)

       `_enqueue_packet(queue: asyncio.Queue, drop_counter: list, pkt) -> None`:
       - This runs in the EVENT LOOP thread (called via call_soon_threadsafe)
       - Calls `make_packet_event(pkt)` to create the event dict
       - If `queue.full()`: call `queue.get_nowait()` to drop oldest (CAP-06), increment `drop_counter[0]`, call `logging.info("Packet dropped (queue full). Total drops: %d", drop_counter[0])` (CAP-07)
       - Then `queue.put_nowait(event)`

       `make_packet_handler(loop: asyncio.AbstractEventLoop, queue: asyncio.Queue, drop_counter: list) -> Callable`:
       - Returns inner function `handler(pkt)` that calls `loop.call_soon_threadsafe(_enqueue_packet, queue, drop_counter, pkt)`
       - This handler is passed to Scapy's `sniff(prn=handler)`

    2. Create `pnpg/capture/interface.py` per RESEARCH.md Pattern 4 (CAP-04, CAP-09):

       `select_interface(config: dict) -> str`:
       - If `config["interface"]` is not None: return `config["interface"]` directly (CAP-09 override). Log INFO with the override name.
       - Otherwise call `psutil.net_io_counters(pernic=True)`
       - If empty dict: raise `RuntimeError("No network interfaces found via psutil")`
       - Sort by `bytes_sent` descending, pick the top one
       - Log INFO: `"Auto-selected interface: %s (bytes_sent: %d)"` (CAP-04)
       - Attempt to map friendly name to Scapy interface via:
         ```python
         try:
             from scapy.config import conf
             scapy_iface = conf.ifaces.dev_from_networkname(iface_name)
             return scapy_iface
         except Exception:
             logging.warning("Could not map '%s' to Scapy interface name, using as-is", iface_name)
             return iface_name
         ```

    3. Create `pnpg/capture/sniffer.py` (CAP-01, partial CAP-10 — supervisor is in Plan 03):

       `start_sniffer(iface: str, packet_handler: Callable, stop_event: threading.Event) -> threading.Thread`:
       - Creates a daemon thread running `_sniffer_target(iface, packet_handler, stop_event)`
