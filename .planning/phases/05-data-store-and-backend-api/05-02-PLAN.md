    **Step 4: Create `pnpg/api/auth.py`.**
    - Suppress passlib bcrypt warning: `import warnings; warnings.filterwarnings("ignore", ".*error reading bcrypt version.*")`
    - `pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")` from `passlib.context`

    **Function `get_current_user`:** FastAPI `Depends` that:
    1. Extracts `HTTPAuthorizationCredentials` from `security`
    2. If credentials is None, raise HTTPException(401, "Missing authorization header")
    3. Calls `jwt.decode(credentials.credentials, config["jwt_secret"], algorithms=["HS256"])` from `jose`
    4. Returns `payload["sub"]`
    5. On `JWTError` or `ExpiredSignatureError`, raises 401

    **Endpoint `POST /auth/setup`:** (AUTH-04)
    1. Check if `app.state.needs_setup` is True; if False, return 409 "Already configured"
    2. Hash password with `pwd_context.hash(body.password)`
    3. Write `{"hash": hashed}` to `Path(config["auth_file"])` (create parent dirs)
    4. Set `app.state.password_hash = hashed`, `app.state.needs_setup = False`
    5. Return `{"data": {"message": "Password set successfully"}}`

    **Endpoint `POST /auth/login`:** (AUTH-01)
    1. If `app.state.needs_setup`, return 503 "Run setup first"
    2. Verify `pwd_context.verify(body.password, app.state.password_hash)`
    3. On failure: raise 401 "Invalid password"
    4. Create access_token: `jwt.encode({"sub": "pnpg-user", "iat": now, "exp": now + timedelta(hours=config["jwt_expiry_hours"])}, config["jwt_secret"], algorithm="HS256")`
    5. Create refresh_token: `jwt.encode({"sub": "pnpg-user", "iat": now, "exp": now + timedelta(days=30), "type": "refresh"}, config["jwt_secret"], algorithm="HS256")`
    6. Return `{"data": {"access_token": ..., "refresh_token": ..., "token_type": "bearer"}}`

    **JWT secret loading** (add to module level or init):
    - If `config["jwt_secret"]` is non-empty, use it
    - Else check env var `PNPG_JWT_SECRET`
    - Else check `data/secrets.json` for key "jwt_secret"
    - Else auto-generate `secrets.token_hex(32)`, write to `data/secrets.json`, log warning "JWT secret auto-generated"
    - Expose as function `resolve_jwt_secret(config: dict) -> str` that returns the secret and mutates config["jwt_secret"] in place
