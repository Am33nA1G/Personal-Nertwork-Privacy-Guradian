    Named SQL constants with `$N` positional parameters. Must include:
    - `INSERT_CONNECTION` — INSERT INTO connections (...21 cols...) VALUES ($1..$21) ON CONFLICT (event_id) DO NOTHING

    - `UPSERT_PROCESS` — INSERT INTO processes (process_name, process_path, first_seen, last_seen) VALUES ($1, $2, $3, $3) ON CONFLICT (process_name) DO UPDATE SET process_path = EXCLUDED.process_path, last_seen = EXCLUDED.last_seen

    - `COUNT_CONNECTIONS` — same WHERE clause, returns COUNT(*)
    - `SELECT_ALERTS` — paginated with optional filters: status ($1::text), severity ($2::text), from_ts ($3::timestamptz), to_ts ($4::timestamptz), LIMIT $5 OFFSET $6
    - `COUNT_ALERTS` — same WHERE, returns COUNT(*)
    - `STATS_SUMMARY` — aggregates for last 24h: total_connections, unique_destinations (COUNT DISTINCT dst_ip), active_alerts (WHERE status='active'), top processes (GROUP BY process_name ORDER BY count DESC LIMIT 10)
    - `STATS_TIMESERIES` — date_trunc($1, timestamp) as bucket, COUNT(*) as count WHERE timestamp >= $2 AND timestamp <= $3 GROUP BY bucket ORDER BY bucket
    - `PURGE_CONNECTIONS` — DELETE FROM connections WHERE timestamp < NOW() - ($1 || ' days')::interval LIMIT 10000 (use a CTE: WITH deleted AS (SELECT event_id FROM connections WHERE timestamp < NOW() - make_interval(days => $1) LIMIT 10000) DELETE FROM connections WHERE event_id IN (SELECT event_id FROM deleted))
    - `PURGE_ALERTS` — same pattern for alerts with separate retention param
    - `SELECT_ALLOWLIST` — SELECT * FROM allowlist ORDER BY created_at DESC
    - `INSERT_ALLOWLIST` — INSERT INTO allowlist (process_name, dst_ip, dst_hostname, expires_at, reason) VALUES ($1, $2, $3, $4, $5) RETURNING *
    - `DELETE_ALLOWLIST` — DELETE FROM allowlist WHERE rule_id = $1 RETURNING *
    - `SELECT_SUPPRESSIONS` — SELECT * FROM suppressions ORDER BY created_at DESC
    - `INSERT_SUPPRESSION` — INSERT INTO suppressions (rule_id, process_name, scope, reason, alert_id) VALUES ($1, $2, $3, $4, $5) RETURNING *

    **Step 2: Create `pnpg/scheduler.py`.**

    ```python
    import asyncio
    import logging
    import time
    from apscheduler.schedulers.asyncio import AsyncIOScheduler

    logger = logging.getLogger(__name__)

    async def purge_old_data(db_pool, config: dict) -> None:
        if db_pool is None:
            return
        conn_days = config.get("retention_connections_days", 30)
        alert_days = config.get("retention_alerts_days", 90)
        async with db_pool.acquire() as conn:
            # Chunked delete for connections
            deleted = 1
            while deleted > 0:
                result = await conn.execute(PURGE_CONNECTIONS, conn_days)
                deleted = int(result.split()[-1]) if result else 0
            # Chunked delete for alerts
            deleted = 1
            while deleted > 0:
                result = await conn.execute(PURGE_ALERTS, alert_days)
                deleted = int(result.split()[-1]) if result else 0
        logger.info("Purge complete: connections > %d days, alerts > %d days", conn_days, alert_days)

    def log_metrics(drop_counter: list[int], last_count: list[int], last_time: list[float]) -> None:
        now = time.monotonic()
        elapsed = now - last_time[0]
        current = drop_counter[0]
        drops_since = current - last_count[0]
        last_count[0] = current
        last_time[0] = now
        logger.info(
            "OBS-04 metrics — drops_since_last: %d, total_drops: %d, interval: %.1fs",
            drops_since, current, elapsed,
        )

    def setup_scheduler(db_pool, config: dict, drop_counter: list[int]) -> AsyncIOScheduler:
        scheduler = AsyncIOScheduler()
        purge_hour = config.get("purge_schedule_hour", 2)
        scheduler.add_job(
            purge_old_data, "cron", hour=purge_hour, minute=0,
            args=[db_pool, config], id="nightly-purge",
        )
        last_count = [0]
        last_time = [time.monotonic()]
        scheduler.add_job(
            log_metrics, "interval", seconds=5,
            args=[drop_counter, last_count, last_time], id="obs-04-metrics",
        )
        scheduler.start()
        return scheduler
    ```
